# Haotian Zhou - Personal Portfolio

This is the repository for my personal portfolio website, hosted at [hati-z.github.io](https://hati-z.github.io).

## Overview

A clean, modern, and responsive personal website designed to showcase my experience, education, and selected work. The site features a dark-themed aesthetic with glassmorphism effects and subtle animations.

## Features

- **Modern Design**: Dark mode with neon accents and glassmorphism UI elements.
- **Responsive Layout**: Optimized for both desktop and mobile viewing.
- **Personal Touch**: Includes a custom "Hati-Z" watermark and specialized timeline view for experience.
- **Performance**: Built with vanilla HTML/CSS/JS for maximum speed and minimal overhead.

## Tech Stack

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom properties (variables), Flexbox/Grid layouts, and CSS animations.
- **JavaScript**: Simple DOM manipulation for smooth interactions.

## Local Development

To run this project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/hati-z/hati-z.github.io.git
    ```

2.  Navigate to the project directory:
    ```bash
    cd hati-z.github.io
    ```

3.  Open `index.html` in your preferred web browser, or serve it using a simple HTTP server (e.g., Python):
    ```bash
    python3 -m http.server 8000
    ```
    Then visit `http://localhost:8000`.
